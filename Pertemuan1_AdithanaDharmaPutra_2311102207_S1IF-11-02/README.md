# Tema Ramadan - Web Bootstrap

Proyek web bertema Ramadan dengan desain menarik menggunakan Bootstrap 5, menampilkan jadwal shalat, countdown, dan informasi THR dalam modal window.

## Fitur

- **Jadwal Shalat** - Tampilan waktu shalat Subuh, Dzuhur, Ashar, Maghrib, dan Isya
- **Countdown Ramadan** - Hitung mundur hari, jam, dan menit menjelang akhir Ramadan
- **Modal THR** - Popup window berisi informasi THR sesuai perintah tugas
- **Desain Responsif** - Optimal di semua ukuran perangkat
- **Tema Ramadan** - Warna dan dekorasi bertema Ramadan yang elegan
